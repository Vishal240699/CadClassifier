The dataset for this project is hosted by Microsoft. To download the necessary dataset for this project, please follow the instructions below.

Go to https://www.microsoft.com/en-us/download/details.aspx?id=54765

Within the unzipped folder, there is a ‘PetImages’ folder. Copy the entire ‘PetImages’ folder into the ‘Dataset’ folder in your project folder. 
   The folder structure should be ‘Chapter04/Dataset/PetImages/’
